# Graph Puzzle Game

This is a graph-based strategy puzzle game where players are challenged to maximize their score by smartly traversing nodes and edges under specific constraints. Each node in the graph holds a point value, and each edge can only be used **once** per submission â€” can you find the optimal path?

---

## Game Objective

Traverse the graph in such a way that you collect the **maximum possible score**, visiting each edge at most once. Strategy, pattern recognition, and optimization are key to climbing the leaderboard.
